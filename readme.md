> Harbergerd is a dapp-custom rollup for dynamically pricing property rights
> using Harberger taxes.

## Prerequisites

- Have [dapptools](https://github.com/dapphub/dapptools#installation) installed

## Run Tests

```bash
dapp test -vv
```
